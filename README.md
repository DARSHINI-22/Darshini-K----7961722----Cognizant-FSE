# Cognizant Digital Nurture 5.0 – Java FSE

This repository contains weekly assignments, hands-on exercises, practice programs, and learning activities completed as part of the Cognizant Digital Nurture 5.0 Java Full Stack Engineer (FSE) curriculum.

## Repository Structure

* Mandatory Assignments
* Weekly Exercises
* Design Pattern Implementations
* SQL Programming Exercises
* Hands-on Activities
* Practice Programs

## Objectives

* Strengthen Core Java concepts
* Understand Design Patterns
* Practice SQL and Database Programming
* Develop problem-solving skills
* Complete curriculum-based hands-on exercises

## Progress

Weekly work and assignments will be updated regularly throughout the program.
